# ComfyUI Custom Scripts and Images

This repository contains custom scripts and images for ComfyUI.

## Contents

### Scripts
- `anya_workflow.json`: Workflow configuration for Anya.
- `download_lora.py`: Script to download LoRA models.
- `download_models.ps1`: PowerShell script to download models.
- `prompt.json`: Prompt configuration.
- `robust_download.ps1`: Robust download script for models.
- `folder_paths.py`: Modified folder paths configuration.

### Images
The `images/` directory contains various PNG images used or generated by the scripts.
